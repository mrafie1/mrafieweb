My Projects

Top Hoops: NBA Optimal Basketball Lineup

I developed the program Top Hoops for a CSC111 group project. The program requires the user to input a dataset of NBA players and their respective statistics. Then it outputs a heatmap of passes between players and a graph displaying the interactions between players, their 'teamwork strength', and the optimal 5-player lineup based on their statistics.
To learn more and try the program out, check out Top Hoops' GitHub page

SnapBack

SnapBack was developed during the RCFTA Hack the Future Hackathon as application to improve the customer return process, reduce overall returned items for businesses, and reducing the environmental impact of returns. My group's product featured 24/7 customer service chat and automatic image detection for item returns: all powered with Google Gemini API.
To learn more, check out SnapBack's GitHub page
